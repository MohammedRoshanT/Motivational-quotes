# Motivational-code
Write Any Motivational Quotes that inspire others and participate in [Hacktoberfest](https://hacktoberfest.digitalocean.com/)    

# Steps to contribute

  #### 1) Fork this repo by clicking fork button on right corner  
  
  ![fork](https://user-images.githubusercontent.com/76490354/137516554-f7a7dff0-8f19-4698-bb00-28635848bb28.png) 
  
  
  #### 2) Click on add file, then click on create new file option
  
 ![add file](https://user-images.githubusercontent.com/76490354/137517974-d3319d31-d702-4e41-b6e7-064b05839628.png)   
![addfile](https://user-images.githubusercontent.com/76490354/137518127-4846064b-f1cf-4269-b48b-2813ebc06af9.png)


  #### 3) Provide name of your file  
  
  ![name](https://user-images.githubusercontent.com/76490354/137518585-1eb3b041-ac5a-4897-a0d0-954d44296dcd.png)
  
  
#### 4) Add content (quotes) to file  

  ![content](https://user-images.githubusercontent.com/76490354/137518865-d2425010-6aa6-4580-a483-bacdd896a841.png)
  
  
#### 5) Click on Commit new file at bottom  

  ![commit](https://user-images.githubusercontent.com/76490354/137519177-0f6df63f-216e-4a2d-b33c-f24e79217efd.png)  
  
  
  #### 6) Click on contribute option in the code section then click open pull request  
  
  ![contribute](https://user-images.githubusercontent.com/76490354/137519563-603f20e9-453b-4e5c-bc19-49dd1492d310.png)
  ![openpr](https://user-images.githubusercontent.com/76490354/137519708-dbc63c56-6bc7-4a67-ac51-292209515561.png)  
  
  
  #### 7) Click on Create pull request  
  
  ![createpr](https://user-images.githubusercontent.com/76490354/137520012-030d2388-dae4-4e2c-a617-b203f30a3ce6.png)
  ![pr](https://user-images.githubusercontent.com/76490354/137520371-6324f6d8-b1d4-45b3-8268-fbc8a831d8fe.png)
  
  
  
  ## congratulations You have contributed successfully !!
